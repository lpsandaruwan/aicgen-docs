# Claude Code Configuration Chunks

Assistant-specific chunks formatted for Anthropic's Claude Code.

## Output Format

Claude Code uses:
- `claude.md` - Main instructions file
- `.claude/` directory - Additional configuration
- Optional hooks and sub-agents

## Chunk Categories

- `language/` - Language-specific best practices
- `architecture/` - Architecture patterns and guidelines
- `testing/` - Testing strategies
- `security/` - Security best practices
- `performance/` - Performance optimization
- `database/` - Database guidelines
- `api/` - API design patterns
